# Document root
Web上に公開するディレクトリのルートを指す。
## Default document root
1. Apache
```
/var/www/html
/var/www
C:\Apache24\htdocs
C:\xampp\htdocs
```
2. Nginx
```
/var/www/html
/usr/share/nginx/html
C:\nginx\html
```
3. IIS
```
C:\inetpub\wwwroot
```
4. Lighttpd
```
/var/www/localhost/htdocs
/var/www
C:\LightTPD\htdocs
```
